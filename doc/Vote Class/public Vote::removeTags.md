## public Vote::removeTags

### Description    

```php
public $Vote -> removeTags ( mixed tags )
```

Remove registered tag(s) on this Vote.    


##### **tags:** *mixed*   
They can be added by string separated by commas or an array.    



### Return value:   

*(array)* List of deleted tags.
