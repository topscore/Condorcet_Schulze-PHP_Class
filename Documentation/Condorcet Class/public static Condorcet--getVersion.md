## public static Condorcet::getVersion

### Description    

```php
public static Condorcet::getVersion ( [bool major = false] ) : string
```

Get the library version.
    

##### **major:** *bool*   
* true will return : '2.0'
* false will return : '2.0.0'
    


### Return value:   

*(string)* Condorcet PHP version.
